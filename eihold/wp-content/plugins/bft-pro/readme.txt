Manage mailing lists and autoresponder campaigns right from your Wordpress blog

== Description ==

BroadFast Autoresponder PRO is a plugin for managing mailing lists, sending newsletters, and autoresponder campaigns.

== Installation ==

1. Unzip the contents and upload the entire `bftpro` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "BFT Autoresponder PRO" in your settings page and manage the plugin
