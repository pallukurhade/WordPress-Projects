== Description ==

Imports into your Clipper site coupons stright from iCodes.


== Installation ==

1. Upload the folder 'appthemes-icodes' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Clipper->iCodes" menu, and provide configuration data.
