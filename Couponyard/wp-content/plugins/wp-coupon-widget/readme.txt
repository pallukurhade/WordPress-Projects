WP <strong>Coupon Widget</strong>: Show coupons to relevant users instantly to increase conversions and build your email list.

== Description ==
Show coupons to relevant users instantly to increase conversions and build your email list. You can require users
to share it before their avail their coupons. This will help you get a lot more referral traffic.

= Benefits = 
<ul>
<li>Quick and Easy setup</li>
<li>Custom message & call to action button</li>
<li>Completely customizable - color, text etc</li>
<li>Launguage independent</li>
<li>Can be made sticky at top of float as user scrolls down</li>
<li>Analytics (available on UsersDelight.com for free)</li>
</ul>

= How it works = 
On setup, Plugin sends an AJAX request to <a href="https://www.usersdelight.com">UsersDelight</a> to save settings (saved locally on wordpress as well) and get a free unique API key. Then, an embedded JS file loads the settings with the given API key
and shows coupon widget as you configured it.

If an email is provided in your settings, you can login to UsersDelight for free so that you can look at in depth analytics on user interaction with coupon widget.

== Installation ==

1. Upload the `wp-coupon-widget` folder to the `/wp-content/plugins/` directory
2. Activate the WP Coupon Widget through the 'Plugins' menu in WordPress
3. Configure the plugin by going to the `Coupons` menu that appears in your admin menu (left)
